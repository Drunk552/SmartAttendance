# T9键盘集成指南

## 概述

本文档介绍如何将T9键盘功能集成到智能考勤系统中。T9键盘是一个4x4的硬件键盘，需要映射到PC键盘，并支持T9输入法（智能输入法）功能。

## T9键盘布局

```
┌───────┬───────┬───────┬───────┐
│   1   │ 2 ABC │ 3 DEF │  ESC  │
├───────┼───────┼───────┼───────┤
│ 4 GHI │ 5 JKL │ 6 MNO │MENU ← │
├───────┼───────┼───────┼───────┤
│7 PQRS │ 8 TUV │9 WXYZ │   ▲   │
├───────┼───────┼───────┼───────┤
│  ● #  │  0 _  │   OK  │   ▼   │
└───────┴───────┴───────┴───────┘
```

## 功能映射

| T9按键 | PC键盘映射 | 功能说明 |
|--------|------------|----------|
| 1 | 键码 0x31 | 数字1 |
| 2 ABC | 键码 0x32 | 数字2，对应字母ABC |
| 3 DEF | 键码 0x33 | 数字3，对应字母DEF |
| ESC | 键码 0x1B | 退出键，功能同电脑ESC |
| 4 GHI | 键码 0x34 | 数字4，对应字母GHI |
| 5 JKL | 键码 0x35 | 数字5，对应字母JKL |
| 6 MNO | 键码 0x36 | 数字6，对应字母MNO |
| MENU ← | 键码 0x0D | 菜单键，跳转到主菜单 |
| 7 PQRS | 键码 0x37 | 数字7，对应字母PQRS |
| 8 TUV | 键码 0x38 | 数字8，对应字母TUV |
| 9 WXYZ | 键码 0x39 | 数字9，对应字母WXYZ |
| ▲ | 键码 0x26 | 上方向键，功能同电脑↑ |
| ● # | 键码 0x23 | 井号键，切换输入法 |
| 0 _ | 键码 0x30 | 数字0，空格和下划线 |
| OK | 键码 0x0D | 确认键，功能同电脑回车 |
| ▼ | 键码 0x28 | 下方向键，功能同电脑↓ |

## T9输入法功能

### 支持的模式
1. **数字模式**：直接输入数字
2. **英文模式**：T9英文输入，按单词字母输入
3. **中文模式**：T9中文输入，按拼音输入
4. **符号模式**：输入常用符号

### 输入法切换
- 按 **#键** 循环切换输入模式
- 当前模式在键盘上方显示

### T9输入原理
1. **英文输入**：输入单词的字母对应的数字键，系统自动匹配单词
   - 例如：输入 228 可能匹配 "cat"、"bat" 等
   
2. **中文输入**：输入拼音对应的数字键，系统匹配汉字
   - 例如：输入 64 可能匹配 "你"（ni）
   
3. **候选词选择**：
   - 使用 **↑↓键** 在候选词间导航
   - 按 **OK键** 确认选择
   - 候选词显示在键盘上方

### 2. 编译配置
在 `CMakeLists.txt` 中添加：

### 3. 基本使用

#### 3.1 初始化T9键盘
```cpp
#include "ui/common/t9_keyboard.h"
#include "ui/common/t9_keymap.h"

// 创建T9键盘逻辑实例
T9Keyboard t9_kb;
t9_kb.init();

// 创建T9键盘映射表
T9KeyMap keymap;
keymap.init();
```

#### 3.2 处理硬件键盘事件
```cpp
void handleHardwareKeyEvent(uint32_t pc_keycode, bool pressed) {
    if (!pressed) return;
    
    // 将PC键码转换为T9键码
    T9KeyCode t9_key = keymap.findT9KeyByPCKeycode(pc_keycode);
    
    // 处理T9按键
    t9_kb.handleKeyPress(static_cast<T9Key>(t9_key));
    
    // 获取当前文本
    std::string text = t9_kb.getCurrentText();
    
    // 获取候选词
    auto candidates = t9_kb.getCandidates();
    
    // 更新UI显示
    updateUIText(text);
    updateUICandidates(candidates);
}
```

#### 3.3 创建LVGL T9键盘UI
```cpp
#include "ui/common/ui_t9_keyboard.h"

// 创建T9键盘UI组件
ui_t9_keyboard_t* t9_ui = ui_t9_keyboard_create(parent_obj);

// 设置关联的文本框
ui_t9_keyboard_set_textarea(t9_ui, textarea_obj);

// 设置文本变化回调
ui_t9_keyboard_set_text_changed_cb(t9_ui, on_text_changed, user_data);
```

### 5. 输入法切换逻辑
```cpp
// 处理#键按下（切换输入法）
void handlePoundKey() {
    T9InputMode current = t9_kb.getCurrentMode();
    T9InputMode next;
    
    switch (current) {
        case T9InputMode::MODE_NUMERIC:
            next = T9InputMode::MODE_ENGLISH;
            showMessage("切换到英文模式");
            break;
        case T9InputMode::MODE_ENGLISH:
            next = T9InputMode::MODE_CHINESE;
            showMessage("切换到中文模式");
            break;
        case T9InputMode::MODE_CHINESE:
            next = T9InputMode::MODE_SYMBOL;
            showMessage("切换到符号模式");
            break;
        case T9InputMode::MODE_SYMBOL:
            next = T9InputMode::MODE_NUMERIC;
            showMessage("切换到数字模式");
            break;
    }
    
    t9_kb.switchInputMode(next);
    t9_kb.clearInput();
}
```

## 故障排除

### 常见问题

1. **按键无响应**
   - 检查键码映射是否正确
   - 验证事件处理函数是否被调用
   - 检查T9键盘逻辑初始化

2. **输入法不切换**
   - 检查#键的键码映射
   - 验证输入法切换逻辑
   - 检查模式显示更新

3. **候选词不显示**
   - 检查字典初始化
   - 验证T9输入处理逻辑
   - 检查UI更新函数

4. **内存泄漏**
   - 确保正确释放T9键盘资源
   - 检查LVGL对象生命周期
