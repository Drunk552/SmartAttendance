### 3. 自助考勤终端的考勤计算流程是什么？

答：考勤计算流程如下：

```mermaid
graph TD
    A((开始)) --> B[读取Excel排班记录]
    B --> C{是否存在excel<br>排班记录}
    
    C -- Yes --> D[考勤计算]
    C -- No --> E{【考勤规则】<br>是否按个人方式<br>排班}
    
    E -- Yes --> F[读取个人排班<br>情况]
    F --> G{是否存在<br>个人排班情况}
    
    E -- No --> H[读取用户部门<br>信息]
    H --> I[读取部门排班<br>情况]
    I --> J{是否存在部<br>门排班记录}
    
    G -- Yes --> K[【考勤规则】<br>读取星期六是否上班规则<br>读取星期日是否上班规则]
    G -- No --> L[【考勤规则】<br>读取默认班次]
    
    J -- Yes --> K
    J -- No --> L
    
    L --> K
    K --> D
    
    D --> M((结束))
